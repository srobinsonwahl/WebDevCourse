(a) If the conversion type is not selected, let the user know to that a conversion type should be selected and set the two values to 0.0 (5).

(c) Show the value that the user entered in the first text box with appropriate units (km or miles)(3).
(d) Show the computed value in the second box with the appropriate units (km or miles). To convert Km to Miles, multiply by (2/3). To go the other way, multiply by (3/2)
